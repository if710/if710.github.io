# IF710 - Programação para Dispositivos Móveis

## 14.08.2019 - Layout Managers e Widgets de UI

### Objetivo

O objetivo desta aula é fornecer uma visão básica do widget `ConstraintLayout`, discutir o papel dos _containers_ em Android, e apresentar alguns outros widgets de UI, bem como mais recursos da linguagem Kotlin.
