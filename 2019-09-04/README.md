# IF710 - Programação para Dispositivos Móveis

## 04.09.2019 - Permissions, Data Management & Room

### Objetivo

O objetivo desta aula é fornecer uma visão de como persistir dados na memória, usando API `File` para salvar arquivos, o mecanismo de `SharedPreferences` e o banco de dados SQLite, com ênfase na biblioteca `Room`.
