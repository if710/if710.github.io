# IF710 - Programação para Dispositivos Móveis

## 20.09.2019 - Princípios de UI Design para Apps

### Objetivo

O objetivo dessa aula é transformar a maneira como aprendemos e percebemos design através de novas visões e reflexões com aplicativos do dia a dia. Analisamos e discutimos apps e suas respectivas interfaces, elementos e funcionalidades numa perspectiva voltada para estudos de UI e UX.
