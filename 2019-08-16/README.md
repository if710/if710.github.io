# IF710 - Programação para Dispositivos Móveis

## 14.08.2019 - RecyclerView

### Objetivo

O objetivo desta aula é fornecer uma visão dos widgets que permitem exibir coleções de dados, como `ListView` e `RecyclerView`. Discutimos o papel de objetos `Adapter` na exibição de informações na tela, e como problemas de desempenho levaram a adoção do padrão `ViewHolder` em widgets `RecyclerView`.
